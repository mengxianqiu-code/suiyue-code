# Run and deploy your app

This contains everything you need to run and share your app.

Live link (GitHub Pages): https://mengxianqiu-code.github.io/suiyue-code/

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deploy

Push changes to the `main` branch. GitHub Actions will build with `npm run build` and publish `dist/` to GitHub Pages automatically.
